# UnusPay Crypto Payments for Easy Digital Downloads

Accept 1000+ cryptocurrencies on your Easy Digital Downloads store with the UnusPay payment gateway. BTC, ETH, USDT, SOL, and meme coins — all supported.

**WordPress.org Plugin Page:** [unuspay-crypto-payments-for-easy-digital-downloads](https://wordpress.org/plugins/unuspay-crypto-payments-for-easy-digital-downloads/)

## Features

- **1000+ cryptocurrencies** — BTC, ETH, USDT, SOL, Doge, Pepe, and more
- **10+ blockchains** — Ethereum, Solana, Tron, Polygon, BSC, Base, Arbitrum, Optimism, Avalanche, and more
- **100+ wallets** — MetaMask, Trust Wallet, Coinbase Wallet, Phantom, and more
- **Instant token swaps** — Automatic conversion between tokens
- **Flat 1% fee** — No hidden charges, no middlemen
- **Direct to your wallet** — Payments go straight to your own wallet, no custody

## Installation

1. Install and activate [Easy Digital Downloads](https://wordpress.org/plugins/easy-digital-downloads/)
2. Download from [WordPress.org](https://wordpress.org/plugins/unuspay-crypto-payments-for-easy-digital-downloads/) or install via Plugins → Add New
3. Go to Downloads → Settings → Payment Gateways → UnusPay
4. Enter your UnusPay API credentials from [unuspay.com](https://unuspay.com)

## Development

This repo mirrors the WordPress.org SVN structure. All plugin code lives in `trunk/`.

```bash
# Edit files in trunk/, then:
git add trunk/ && git commit -m "fix: description" && git push origin main
# → CI automatically syncs trunk/ to WordPress.org SVN
```

### Release a new version

```bash
# Trigger CD pipeline (auto-increments version, creates SVN tag)
gh workflow run release.yml

# Or dry-run first
gh workflow run release.yml --field dry_run=true
```

### Versioning

- Patch auto-increments: `1.0.0` → `1.0.1` (caps at 10)
- At patch 10, minor bumps: `1.0.10` → `1.1.0`
- Major version is always manual

## Project Structure

```
trunk/               ← Plugin code (edit here)
  ├── *.php           ← Main plugin file (870 lines, procedural PHP)
  ├── assets/         ← CSS, JS, images
  └── readme.txt      ← WordPress.org metadata
assets/               ← WordPress.org banners & screenshots (top-level)
tags/                 ← SVN release snapshots (auto-generated by CD)
scripts/release.sh    ← Release automation script
.github/workflows/    ← CI/CD pipelines
```
